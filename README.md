# zeanium-web
Zeanium for web




## Installation

```bash
npm install @zeanium/web -s
```

## Usage

### Cookie

You can call `zn.cookie` to use Cookie feature.
The API as below:

|Method|Description|
|:--:|:----------:|
|**`zn.cookie.clear()`**|Clear all cookie value.|



Here's an example code how to use cookie

```javascript
    
    zn.cookie.clear();

```
